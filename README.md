# 🌐 Web Basics
이 레포는 **웹 개발 기초(HTML, JavaScript, Node.js)** 학습 내용을 정리한 저장소입니다.

---

## 🗂️ 학습 목표
- HTML 태그와 문서 구조 이해
- JavaScript 문법 및 DOM 조작 학습
- Node.js로 간단한 서버 구현

---

## 📑 학습 내용
- **HTML**
  - 기본 태그, 폼, 테이블
  - 시맨틱 태그
- **JavaScript**
  - 변수, 함수, 제어문
  - DOM 접근 및 이벤트 처리
- **Node.js**
  - 간단한 서버 생성
  - 라우팅 및 모듈 사용
